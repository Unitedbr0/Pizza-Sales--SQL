# Pizza Sales Analysis Using SQL

## Project Overview
This project focuses on analyzing pizza sales data using SQL to extract meaningful business insights.  
The analysis helps understand customer ordering behavior, sales performance, and revenue trends.

## Objectives
- Analyze total number of orders placed  
- Calculate total revenue generated  
- Identify most popular pizza sizes and types  
- Find highest revenue-generating pizzas  
- Analyze category-wise and time-based sales distribution  
- Track cumulative revenue growth over time  

## Dataset Description
The project uses the following CSV files:
- **orders.csv** – Order date and time details  
- **order_details.csv** – Pizza quantities per order  
- **pizzas.csv** – Pizza size and price information  
- **pizza_types.csv** – Pizza names and categories  

## Tools & Technologies
- SQL (MySQL)  
- Relational Database Concepts  
- Joins, Aggregate Functions, Window Functions  
- GitHub for version control  

## Key Analysis Performed
- Total orders and total sales revenue  
- Top-selling pizza types and sizes  
- Revenue contribution by pizza category  
- Hourly order distribution  
- Category-wise top pizzas based on revenue  
- Cumulative revenue analysis over time  

## Key Insights
- Certain pizza categories contribute more to overall revenue  
- Large-sized pizzas are the most frequently ordered  
- Peak orders occur during specific hours of the day  
- A few pizza types generate a significant portion of total sales
